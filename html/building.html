Building rosjava_core¶

rosjava_core uses the Gradle build system. To build rosjava_core, execute the gradle wrapper:

Note that the build process currently involves extra steps that will be folded into Gradle tasks.

  1. roscd rosjava_core
  2. ./gradlew rosjava_bootstrap:install
  3. rosrun rosjava_bootstrap install_generated_modules.py rosjava
  4. ./gradlew rosjava:install

Then, for each rosjava_core package you’re interested in (e.g. foo):

  1. rosrun rosjava_bootstrap install_generated_modules.py foo
  2. ./gradlew foo:install

See the rosjava Javadoc to learn more about the API.

Note that package level documentation is in progress.
