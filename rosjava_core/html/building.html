Building rosjava_coreÂ¶

rosjava_core uses the Gradle build system. rosmake is not supported.

To build rosjava_core, execute the gradle wrapper:

Note that the build process currently involves extra steps that will be folded into Gradle tasks or otherwise eliminated.

  1. Install Maven 3.
  2. roscd rosjava_core
  3. ./gradlew rosjava_bootstrap:install
  4. rosrun rosjava_bootstrap install_generated_modules.py rosjava
  5. ./gradlew rosjava:install

Then, for each rosjava_core package you’re interested in (e.g. foo):

  1. rosrun rosjava_bootstrap install_generated_modules.py foo
  2. ./gradlew foo:install

To generate Eclipse project files:

  1. ./gradlew foo:eclipse
