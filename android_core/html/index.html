android_core¶

android_core is a collection of components and examples that are useful for developing ROS applications on Android. It provides a base Activity (RosActivity) and Service (NodeMainExecutorService) for executing and managing the lifecycle of your NodeMains.

In addition, android_core defines the pattern of combining the Android View and NodeMain concepts that enables the development of data driven Android UIs. Several such RosViews (e.g. RosTextView, RosImageView, and RosCameraPreviewView) are provided.

Support is best found on http://answers.ros.org/.

Please file bugs and feature requests on the rosjava issues page. Starring issues that are important to you will help developers prioritize their work.

In addition to the following documentation, rosjava_core makes liberal use of Javadoc.
